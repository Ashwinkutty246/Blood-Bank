dnacreative's sandbox: BloodBank Distro

Primary tabs

  • View(active tab)
  • Version control
3 people starred this project

Experimental project

This is a sandbox project, which contains experimental code for developer use only.

This is the implementation of a Blood Bank management system based on Drupal 7 CMS. It was developed for the needs on an Research Program in collaboration with a Greek Hospital and the Midwifery Department of TEI.

Supporting organizations: 
DNA Creative Group LTD
